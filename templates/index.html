The idea is to automate prompt engineering, using pre-estabilished (and very much thought-out) prompts to enhance results. Instead of making templates and having humans manually edit prompts, why not have GPT-3 do it by itself in varying levels of complexity? These are the options available:
  • Image Generation: Write an idea, as short or long as you like, of an image you want to see (E.g.: Flying horsemen). It will return a better prompt that you can use to generate high-quality images in Image Generation models
  • Historical Text: Write the name of a historical(or current) civilization(E.g.: Assyria). It will generate a text on a random subject in the style of the selected civilization and reflecting its values. Threee different OpenAI API calls are involved in generating the final prompt
  • Scientific Article: Write a subject or field of study (E.g.: Neuroscience). The app will then ask GPT-3 to generate a prompt, and use that generated prompt for the final API call
